# CyberForge - Advanced Security Toolkit
# Requirements: Python 3.8+ (NO external dependencies needed!)
#
# CyberForge uses only the Python standard library.
# No pip install required!
#

#   whois    - WHOIS lookups
#   dig      - DNS reconnaissance  
#   nmap     - Advanced port scanning
#   nmcli    - WiFi scanning (Linux)
#
# Install on Debian/Ubuntu:
#   sudo apt install whois dnsutils
#
# Install on macOS:
#   brew install whois bind
